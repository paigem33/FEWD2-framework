Classes and what they do:

Container/Grid

Use the class of container to show where the bulk of your page content will be. Think of it as a tag to wrap around everything on the page. If you want the entire page to be fixed in the middle of the screen at the standard width of 960px, use the class of container along with the class of small. For a width of 1024px, add class="container medium" to your HTML element.

We use flexbox for our grid system. To use it, you will have to have a parent element with the class of flex, and then based on how many columns you want, you will have col-1, col-2, col-3, or col-4 on each of the child elements. The idea here is that you don't have to worry about screen sizes. On smaller mobile devices, each column type will take up the full width. The maximum number of columns will depend on the screen width. For example, col-4 makes the number of columns go from a single column on a small screen, to two columns (side by side) on medium size screens, to four columns on larger screens.

Another variation is to have a smaller column taking up about one third of the container width (or roughly 33% width, hence the class name col-33) and/or a larger column taking up about two thirds (about 66%) width: col-66.

You can make buttons by either using the <button> tag or by adding the .button class to an <a> tag. There are different sizes and different colors built right in. See below for examples and what classes you'll need to use for each.

Misc/helper classes

We have a .float-left class to quickly float an item left, a .float-right class to quickly float an item right, and a .clearfix class to quickly clear items from floating.

Use the .no-print class to keep content from being printed

To quickly make an image responsive, use the .responsiveImg class.

To quickly make an image or any element border rounded around, use the .border-rounded class.

If you want an element to have a full width, use the .full-width class.
