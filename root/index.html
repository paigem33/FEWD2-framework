Classes and what they do:

Container/Grid

Use the class of container to show where the bulk of your page content will be. Think of it as a tag to wrap around everything on the page. If you want the entire page to be fixed in the middle of the screen at the standard width of 960px, use the class of container along with the class of small.

We use flexbox for our grid system. To use it, you will have to have a parent element with the class of flex, and then based on how many columns you want, you will have col-1, col-2, col-3, col-4 on the child elements. You also have the option to have some of the columns larger or smaller than the others using col-33 or col-66. These are demonstrated below:

Forms

You can use any form element, use the .group-form class to group the specific form elements you want to use, and .form-control class to add style to the elements

Misc/helper classes

We have a .float-left class to quickly float an item left, a .float-right class to quickly float an item right, and an .clearfix class to quickly clear items from floating.

We have a .no-print class to keep content from being printed

We have .responsiveImg class to quickly make an image responsive.
